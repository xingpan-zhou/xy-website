## 图片优化建议

1. **格式选择**：
   - 照片用 JPG
   - 图标、Logo 用 PNG
   - 动画用 GIF 或 WebP

2. **尺寸优化**：
   - 头像：400x400 px
   - 项目截图：800x600 px
   - 背景图：1920x1080 px

3. **文件命名规范**：
   - 使用小写字母
   - 用连字符分隔：`my-project.jpg`
   - 避免空格和特殊字符
